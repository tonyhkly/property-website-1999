In the winter of 2019, we took part in a work hackathon project. Our aim? To create the tackiest, weirdest, funnest, headache inducing property website that has ever been made! We wanted to use every website nostalgic cliche from the 90's and 00's to make everyone's head explode!!! The team was made up of 3 engineers and 7 progamming newbies from various parts of the company including Customer Support and Business Analysis. We wanted to create a fun website, but also encourage people learn to code even if they've never opened up a code editor before. You've looked through our website now so I'll leave it up to you to decide if we suceeded!
